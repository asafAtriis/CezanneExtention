# Cezanne HR Time Auto-Filler
 
## Local Installation Guide - Steps:
1. **Download** the extension zip file and unzip it to a folder
 
2. **Open** `chrome://extensions` in Chrome Or `edge://extensions` in Edge
 
3. **Enable** "Developer mode" (top-right toggle)
 
4. **Click** "Load unpacked" → select the extension folder
 
5. **Pin** the extension icon to toolbar (optional)
 
## Usage
 
1. Go to Cezanne HR → Clock In / Out page
2. Click extension icon
3. Set dates and times
4. Click "Start Auto-Fill"
